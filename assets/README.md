# Game Assets

## Images Needed

Place these images in the `images/` folder:

1. **godzilla_idle.png** - Godzilla standing/waiting animation
2. **godzilla_stomp.png** - Godzilla stomping animation
3. **s3_bucket.png** - AWS S3 bucket icon (glowing cyan style)
4. **background.png** - Cyberpunk/sci-fi background
5. **data_center.png** - AWS data center building

### Image Specifications
- Format: PNG with transparency
- Size: 512x512px recommended for sprites
- Style: Cyberpunk aesthetic with cyan/blue glows

## Sounds Needed

Place these sounds in the `sounds/` folder:

1. **stomp.mp3** - Successful bucket stomp sound
2. **miss.mp3** - Miss/error sound
3. **godzilla_roar.mp3** - Godzilla roar for round starts
4. **combo.mp3** - Combo achievement sound

### Sound Specifications
- Format: MP3 or OGG
- Length: 0.5-2 seconds for effects
- Volume: Normalized to prevent clipping
